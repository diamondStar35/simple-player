Simple Audio Player User Guide (Version 1.0.1)

Last updated: March 7, 2026

This guide explains the app in plain language for day-to-day use. It starts with the interface, then goes through playback and file workflows, then covers YouTube, recording, settings, shortcuts, and troubleshooting. The goal is to explain how features are actually used in normal listening, not just list menu labels.

Overview

Simple Audio Player is a Windows media player built around fast keyboard control and predictable behavior. You can use it for local files, folder-based playlists, stream links, YouTube videos/playlists, and recording. The app keeps most actions visible in the main menu bar, so once you understand the menu structure, you can move through features quickly.

The app also separates feature data cleanly. Settings, bookmarks, favorites, and saved playback positions are stored in different files, so each feature can be backed up or reset without affecting everything else.

System Requirements

The app is designed for Windows desktop use. You need a working sound output device (speakers, headphones, or any configured audio output). If you plan to record, you also need a working input device such as a microphone.

Internet is optional for local playback. It is only needed for update checks and YouTube features. If you use the app only for local files, it still works normally without YouTube components.

Installation and First Launch

The simplest way to install is from the project releases page. Use the installer build if you want a normal installed app entry, or use the portable zip build if you prefer to run it from a custom folder.

On first launch, open one local file first to confirm output is correct. Then open settings with Ctrl+P (or from File > Preferences...) and review General and Audio pages. This first setup is where most long-term behavior is defined, especially open-with-files mode, save-on-close behavior, and playback step values.

Main Interface

The main window has transport buttons and a menu bar. The transport buttons are for immediate playback actions (previous, rewind, play/pause, forward, next). The menu bar is where you access full workflows such as bookmarks, favorites, YouTube tools, recording, and preferences.

Menu Bar Structure

  • File: opening files/folders/links, favorites, file list/properties, preferences, close/exit actions.
  • Edit: rename/delete/copy/paste for local files, plus mark controls.
  • Bookmarks: add, manage, and jump-to-slot actions.
  • Actions for marked files: batch copy/move/delete/clipboard actions.
  • Player: playback navigation, seek/time controls, A-B loop, speed, shuffle/repeat, silence removal, sound cards.
  • Video options: YouTube-specific actions for current video context.
  • Recording: recording start/pause/stop and recordings folder.
  • Help: guide, what's new, contact options, updates, and about dialog.

Some menu items become disabled when they do not apply. For example, local file properties and bookmark actions are disabled for URLs and YouTube streams.

Basic Playback

To open one local file, press Ctrl+O, or use File > Open File.... To build a playlist from a folder, press Ctrl+Shift+O, or use File > Open Folder.... To open a direct stream, press Ctrl+L, or use File > Open Link....

Play/pause uses Space or Enter. Next/previous uses Tab/Page Down and Shift+Tab/Page Up. This is usually enough for normal playback.

For navigation inside long files, use left/right arrows for seek, then use modifier variants when needed: Ctrl+Left/Right (x2), Ctrl+Shift+Left/Right (x4), and Shift+Left/Right (x8). You can also jump by percentage with Ctrl+1 to Ctrl+0, and the shifted set for offset jumps.

Exact file index navigation is available through go-to-file: press Ctrl+G, or choose Player > Go to file.... Exact time navigation is available through go-to-time: press Ctrl+Shift+G, or choose Player > Go to time.... The go-to-time dialog validates values against actual duration and keeps the dialog open if the value is invalid.

A-B loop is useful when you need to repeat a segment. Set A with [, set B with ], and clear with Backspace. The same actions are available from Player > A-B loop.

File and Playlist Management

The opened-files dialog is the fastest way to jump in large playlists. Open it with F2, or choose File > Opened Files.... Select an item and press Enter to jump.

For local file operations, use rename with Shift+F2 or Edit > Rename..., and delete with Shift+Delete or Edit > Delete. Open containing folder with Ctrl+F or File > Open Containing Folder. Open file properties with Alt+Enter or File > File properties.... These operations are local-file only by design.

Marking is useful when managing multiple files in one pass. Mark current with Ctrl+K, mark all with Ctrl+A, clear marks with Ctrl+Shift+K, and announce marked count with K. After marking, use Actions for marked files to copy, move, copy paths to clipboard, or delete.

Two audio settings affect playlist behavior heavily. "Wrap to top for multiple files" makes next/previous circular when there is more than one file. "Save current position for each file" saves position when you move between files and restores the saved point when returning to that same file.

Bookmarks

Bookmarks are named positions for local files. They are intended for quick return points in long material such as courses, podcasts, and interviews.

Add a bookmark with Shift+M, or from Bookmarks > Add a new bookmark. The app asks for a name and stores current position with that name.

Open bookmark manager with Ctrl+Shift+M, or from Bookmarks > Manage bookmarks. The dialog shows name and position columns, and provides jump, edit name, and delete actions. Pressing Enter on a selected bookmark jumps directly to its saved position.

Quick slot jumping is available through Alt+1 to Alt+9 and Alt+0 for slot 10. The same actions are available under Bookmarks > Jump to bookmark.

Favorite Videos

Favorite Videos stores reusable links with an explicit type. It is useful when you often open the same playlist, video, livestream, or generic stream URL and want one central place to manage those links.

Open the dialog with Ctrl+Shift+F, or from File > Favorite videos.... The list contains name, type, and link. Use add/edit/remove buttons to maintain the list.

Type behavior is important: "Video" and "Playlist" force their specific YouTube flow, "Combined link" follows your mixed-link setting in YouTube settings, and "Generic stream" opens directly as normal stream without YouTube resolver behavior.

YouTube Features

YouTube support includes direct link opening, search, result browsing, and video-related actions. Open YouTube link with Ctrl+Shift+Y (or File > Open YouTube Link...) and search with Ctrl+Y (or File > Search YouTube...).

Search opens a results dialog where you can play entries, copy links, open browser/channel, and download from result context. Playlist links open a browse flow, while direct video links can be played immediately.

For links that include both video and playlist IDs, behavior is controlled by YouTube settings: ask every time, always video, or always playlist. This setting also affects favorites of type "Combined link".

Video actions for current YouTube context are available through Ctrl+D for download, Alt+D for description, or from the Video options menu.

YouTube components can be updated from Help > Updates > Update YouTube components or from Settings > YouTube. The startup update-check option for yt-dlp compares local and remote versions and shows the regular update prompt when an update is available.

Recording

Recording captures from your default input device and writes to the configured recordings folder. Start with F9, pause/resume with F7, and stop with F8, or use the same commands from the Recording menu.

Start and resume operations use a blocking beep before capture continues. Stop plays a lower-frequency confirmation beep after recording is stopped.

You can open recordings folder directly from Recording > Open recordings folder. Global recording control is available with Alt+Win+F9, Alt+Win+F7, and Alt+Win+F8.

Settings Interface

Open settings with Ctrl+P (or File > Preferences...). Categories are displayed as a flat list on the left, and each category page appears on the right. Pressing OK validates all pages. If a value is invalid, a message box appears and the dialog stays open so you can correct it.

Press F1 while focused on any setting control to hear short help for that specific option. This is useful when you are not sure what a setting changes and want quick guidance without leaving the dialog.

Settings Categories

12.1 General Settings

This page defines how the app behaves across launches. Language, verbosity, and update checks are all managed here, along with the two options that affect daily workflow the most: open-with-files mode and save-on-close. If open-with-files mode is set to include folder content, opening one file can build a full playlist automatically. If save-on-close is disabled, session-level changes are intentionally discarded on exit.

General settings also includes \"Remember last file position\" for startup restore behavior and file-association actions for Explorer integration.

12.2 Backup and Restore

Use this page when moving to another machine, recovering from broken settings, or saving a backup before major changes. You can export and import settings, export and import bookmarks separately, reset settings to defaults, and open the settings folder directly.

Keeping bookmarks import/export separate from settings helps if you want to keep listening markers while testing a fresh settings profile.

12.3 Audio Settings

Audio settings control seek behavior, speed and volume increments, and file-end behavior. This is also where playlist wrapping and per-file position tracking are configured. If you frequently switch between many files, enabling per-file position tracking makes navigation feel much more continuous.

Processing options such as dynamic normalize/limiter and mono playback are also on this page. They are useful for difficult source material, especially content with uneven loudness or channel imbalance.

12.4 Silence Removal Settings

This page controls the FFmpeg silence-removal filter. In basic use, you only need threshold and minimum duration. Advanced controls expose full filter parameters for more precise trimming behavior.

If speech sounds too choppy, reduce aggressiveness by adjusting threshold and duration before touching advanced controls.

12.5 Recording Settings

Recording settings define input channel layout (mono/stereo), bitrate preset, output format (WAV/MP3), and the target recordings folder. Most users only change format and folder, but bitrate presets are useful when you need smaller files for long recordings.

12.6 YouTube Settings

YouTube settings decide how links are handled and how components are updated. Mixed-link behavior is especially important because it controls what happens when one URL contains both a video ID and playlist ID.

The yt-dlp startup-check option compares local and remote versions and triggers the same update flow used by the manual update command when a newer version is detected.

12.7 Keyboard Shortcuts and Global Shortcuts

These two pages are for custom bindings. Keyboard Shortcuts affects focused app controls. Global shortcuts affect actions while the app runs in background. Conflicts are blocked inside each scope, and defaults can be restored from the same page.

Configuring Shortcuts

For local shortcuts, open settings with Ctrl+P, select Keyboard Shortcuts, pick an action, click edit, then press the new combination. For global bindings, follow the same process in Global shortcuts.

If you choose a key combination that is already assigned to another action in the same scope, the app shows a conflict message and keeps the original assignment. You can use "Reset to Defaults" at any time.

Local Shortcuts Reference

Playback and Navigation

  • Space or Enter: Play or pause the current file.
  • Tab or Page Down: Move to the next file in the playlist.
  • Shift+Tab or Page Up: Move to the previous file in the playlist.
  • Ctrl+Home: Jump to the first file in the playlist.
  • Ctrl+End: Jump to the last file in the playlist.
  • Ctrl+G: Open the Go to File dialog.
  • Ctrl+Shift+G: Open the Go to Time dialog.

Seek and Position

  • Left: Seek backward using the current movement value.
  • Right: Seek forward using the current movement value.
  • Ctrl+Left: Seek backward with 2x movement.
  • Ctrl+Right: Seek forward with 2x movement.
  • Ctrl+Shift+Left: Seek backward with 4x movement.
  • Ctrl+Shift+Right: Seek forward with 4x movement.
  • Shift+Left: Seek backward with 8x movement.
  • Shift+Right: Seek forward with 8x movement.
  • Home: Jump to the beginning of the current file.
  • End: Jump to the end of the current file.
  • Shift+1 to Shift+0: Apply built-in seek-step presets.
  • Shift+-: Use the custom seek value from Audio settings.
  • Ctrl+1 to Ctrl+0: Jump to 10% through 100% of the file.
  • Ctrl+Shift+1 to Ctrl+Shift+0: Jump to 15% through 100% offset positions.

Volume and Speed

  • Up: Increase volume.
  • Down: Decrease volume.
  • Shift+Up: Set volume to maximum level.
  • Shift+Down: Set volume to minimum level.
  • V: Announce the current volume.
  • Ctrl+Up: Increase playback speed.
  • Ctrl+Down: Decrease playback speed.
  • Alt+Y: Reset playback speed to normal (1x).
  • S: Announce the current playback speed.

File and Edit

  • Ctrl+O: Open file dialog.
  • Ctrl+Shift+O: Open folder dialog.
  • Ctrl+L: Open link dialog.
  • Ctrl+F: Open containing folder for current local file.
  • Alt+Enter: Open Windows file properties for current local file.
  • F2: Open the Opened Files playlist dialog.
  • Ctrl+W: Close the current file.
  • Ctrl+Shift+W: Close all loaded files.
  • Shift+F2: Rename current local file.
  • Shift+Delete: Delete current local file.
  • Ctrl+Shift+C: Copy current file for paste/file operations.
  • Ctrl+V: Paste files or links from clipboard.
  • Ctrl+K: Mark the current file.
  • Ctrl+A: Mark all files in the playlist.
  • Ctrl+Shift+K: Clear all marked files.
  • K: Announce how many files are marked.

Bookmarks, Favorites, and YouTube

  • Shift+M: Add a bookmark at the current playback position.
  • Ctrl+Shift+M: Open bookmark manager for the current file.
  • Alt+1 to Alt+0: Jump to bookmark slot 1 through 10.
  • Ctrl+Shift+F: Open Favorite Videos dialog.
  • Ctrl+Shift+Y: Open YouTube link dialog.
  • Ctrl+Y: Open YouTube search dialog.
  • Ctrl+D: Start download for current YouTube video context.
  • Alt+D: Open description for current YouTube video context.

Utilities and Recording

  • [: Set A-B loop start point (A).
  • ]: Set A-B loop end point (B).
  • Backspace: Clear A-B loop selection.
  • Ctrl+Z: Toggle shuffle mode on or off.
  • Ctrl+R: Toggle repeat-current-file mode on or off.
  • Ctrl+M: Toggle silence-removal filter on or off.
  • E: Announce elapsed playback time.
  • R: Announce remaining playback time.
  • T: Announce total file duration.
  • P: Announce current playback percentage.
  • F: Announce current file information.
  • Ctrl+Shift+A: Open sound cards dialog.
  • Ctrl+Shift+T: Run speaker test.
  • Ctrl+Shift+V: Toggle verbosity mode.
  • F9: Start recording.
  • F7: Pause or resume recording.
  • F8: Stop recording.

Data, Backup, and Restore

The app keeps feature data in separate files under the same settings directory. The main settings file is settings.ini. Bookmarks are saved in bookmarks.json, favorites in favorites.json, and per-file playback positions in positions.json.

Backup and restore page provides import/export for settings and bookmarks, plus settings reset and quick access to the settings folder.

Troubleshooting

If output is wrong, run speaker test with Ctrl+Shift+T or from File > Test speakers, then check sound cards selection. If go-to-time fails, verify selected time is inside current duration. If bookmarks are disabled, make sure current source is a local file, not a URL/YouTube stream.

If per-file position does not restore as expected, confirm "Save current position for each file" is enabled. If settings are not preserved between launches, confirm "Save settings on close" is enabled in General settings.

For YouTube failures, update components from Help updates submenu or YouTube settings page. For global shortcut issues, rebind to another combination to avoid conflicts with Windows or other apps.

Global Shortcuts

Global shortcuts work while the app is running even when its window is not focused. If one of these shortcuts does not trigger, the most common reason is conflict with another app or a system-level shortcut.

  • Alt+Win+Space: Play or pause the current file globally.
  • Alt+Win+Left: Seek backward globally.
  • Alt+Win+Right: Seek forward globally.
  • Alt+Win+Up: Increase volume globally.
  • Alt+Win+Down: Decrease volume globally.
  • Alt+Win+Page Down: Move to next track globally.
  • Alt+Win+Page Up: Move to previous track globally.
  • Alt+Win+F9: Start recording globally.
  • Alt+Win+F7: Pause or resume recording globally.
  • Alt+Win+F8: Stop recording globally.
